# Interests
Сайт для отслеживания просмотренных фильмов и сериалов, сыгранных игр.

# Сборка и развертывание

Фронтенд лежит в `frontend-new`, а production-артефакты собираются в `frontend-dist`.
Папка `frontend-dist` является generated-контентом и не хранится в git.

Для production-сборки фронтенда используется корневой скрипт:

```bash
./build-frontend.sh
```

После этого можно пересобирать и поднимать backend-контейнеры обычным `docker compose`.
